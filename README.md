# Analoge Uhr mit Dark/Light Mode

Eine stilvolle analoge Uhr in HTML, CSS und JavaScript mit **Dark/Light Mode** und animierten Zeigern.  

## Features

- Analoge Uhr mit Stunden-, Minuten- und Sekundenzeigern
- Dark- und Light-Mode per Button umschaltbar
- Responsive Design
- Dezente Animationen
